Anchor Tags

Use the href attribute with the anchor tag to reference another page. Take me to Google

Code for this Page


        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <title>Anchor Tags</title>
        </head>
        <body>
            <h1>Anchor Tags</h1>
            <p>Take me to <a href="https://www.google.com">Google</a></p>
        </body>
        </html>
