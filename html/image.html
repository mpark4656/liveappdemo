Adding Images to a Webpage

The image tag doesn't have a closing tag. Add an image source by using the src attribute.

Code for this Page


        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <title>Adding Images to a Webpage</title>
        </head>
        <body>
            <h1>Adding Images to a Webpage</h1>
            <p>The image tag doesn't have a closing tag. Add an image source by using the src attribute.</p>
            <img src="img/tree.jpg" alt="Tree">
        </body>
        </html>
