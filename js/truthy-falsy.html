Truthy and Falsy Values

All JS values have an inherent truthyness or falsyness about them.

  • false
  • 0
  • "" (Empty String)
  • null
  • undefined
  • NaN

All other values are truthy.

Below is an example of using string's truthy/falsy nature to evaluate if the input has been provided.


        document.querySelector("#prompt-button").addEventListener("click", event => {
            const input = prompt("Enter something");
            if(input.trim()) {
                alert(`You entered something: ${input.trim()}.`);
            } else {
                alert(`No characters in your input.`);
            }
        })
