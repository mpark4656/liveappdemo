Logical Operators

AND (&&)

Joins left and right boolean operands. Both operands must be true for the operator to return true. AND operator has precedence over OR.

OR (||)

Only one of the operands must be true.

Not (!)

Flips true to false and vice versa. 


    const age = 90;
    if(age < 5 || age >= 65) {
        console.log("FREE");
    } else if(age > 5 && age < 10) {
        console.log("$10");
    } else if(!(age >= 65)) {
        console.log("$20");
    }
