Arrays

There are several methods for adding or removing items in arrays

  • push - add to end
  • pop - remove from end
  • shift - remove from start
  • unshift - add to start

        let movieLine = ['tom', 'nancy', 'pablo', 'oliver', 'eva'];

        // Show the current array
        console.log(movieLine);

        // Remove the first element
        movieLine.shift();

        // Add jerry to the beginning of the array
        movieLine.unshift('jerry');

        // Remove the last element
        movieLine.pop();

        // Add mike to the end of the array
        movieLine.push('mike');
