String

String is another primitive type that represents texts. A string variable can hold an empty value. String values are often surrounded by double or single quotation marks. 


        let doubleQuoteStr = "String with double quotation";
        let singleQuoteStr = 'String with single quotation';
        let backTickStr = `String with backtick marks or template string ${5 + 2}`;

        console.log(doubleQuoteStr);
        console.log(singleQuoteStr);
        console.log(backTickStr);

String is a collection of characters. Each character can be accessed with an index, starting at 0. Strings are immutable, so you can't change or modify any characters in a string. You can only reassign a new value to a string variable. 


        let str = "Test";
        console.log(str[0]);    // T
        console.log(str[1]);    // e
        console.log(str[2]);    // s
        console.log(str[3]);    // t
        str[3] = "T";           // Nope
        console.log(str);       // str still has "Test"

String has a number of methods that allow us to perform certain operations on the string. Syntax: String.methodName(); String variables are immutable, so the original string value does not change. 


        let msg = "Leave me alone";
        console.log(msg.toUpperCase());  // Prints the message in all uppercase
        let strWithSpaces = "    Trim Me!    ";
        console.log(strWithSpaces.trim());  // Spaces before and after the string are trimmed.
