Math Object

The build-in Math object provides some mathematics utilities

  • PI constant
  • Absolute Value
  • Power function
  • Square Root
  • Rounding
  • floor & ceiling
  • Random number
  • Many more...

Math.random() generates a decimal number between 0 (inclusive) and 1 (exclusive).

What if you want a random integer? Just floor the result from Math.random().


        console.log(`The value of PI is ${Math.PI}`);
        console.log(`Floor always rounds down - ${Math.floor(2.67)}`);
        console.log(`Ceiling always rounds up - ${Math.ceil(2.67)}`);
        console.log(`The square root of 4 is ${Math.sqrt(4)}`);

        let randomInteger = Math.floor(Math.random() * 10);
        console.log(randomInteger);
