String Template Literal

Backticks (`) are used to encapsulate strings, but it allows you to interpolate expressions using the ${var} syntax. Inside ${} you can use any javascript expression. This helps avoid repetitive usage of + to concatenate strings that contain variables. 


        let product = "Artichoke";
        let price = 3.99;
        let qty = 5;

        console.log(
            `You bought ${qty} of ${product} at $${price} each. The total price is ${(price * qty).toFixed(2)}.`
        );
