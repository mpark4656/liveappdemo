Comparison Operators

Operators used to compare left operands and right operands

  • Greater than (>)
  • Less than (<)
  • Equal to (==)
  • Greater than or Equal to (>=)
  • Less than or Equal to (<=)

        1 > 3   // false
        1 < 3   // true
        1 <= 1  // true
        1 >= 1  // true
        1 < 1   // false

Other data types can be compared, doesn't always have to be numbers. Strings and characters can be compared. The unicode of the characters will be used for comparison. 


        "test" == "test"   // true

Triple (Strict) vs. Double (Loose) Equality

Double equality checks if two values are equal regardless of the type. Both operands are converted to the same type. 


        1 == 1              // true
        1 == '1'            // true
        null == undefined   // true
        0 == false          // true

        1 === 1              // true
        1 === '1'            // false
        null === undefined   // false
        0 === false          // false

        0 !== '0'   // true
        0 != '0'    // false
