Undefined & Null

Null

  • Intentional absence of any value
  • Must be assigned manually

Undefined

  • Variables that do not have an assigned value are undefined.
  • A function that doesn't have a return statement returns undefined.
  • A variable that hasn't been initialized is undefined.

        let undefinedVar;
        console.log(undefinedVar);  // undefined

        let nullVar = null;
        console.log(nullVar);   // null
