How CSS "Cascades"

The order matters in CSS! If there are two pieces of CSS codes that select the same element and apply different values for the same attribute, the latter will be applied and the former will be overridden.

For example, this button below is applied a green color first and then red. Obviously, the red wins because it was applied after the green.

The same rules are applied on CSS files that are referenced via the link element. If there is a conflict, the attributes from the last added CSS files are applied
