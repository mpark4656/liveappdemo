Flexbox - Align Items

The boxes below are in a row-directional flex box. The items are justified in the center. With the flex-end align-items, the items are pushed to the bottom. The default is flex-start.

If you want to place items centered horizontally and vertically, you can set both justify-content and align-items to center. This is true for both row and column directions.
