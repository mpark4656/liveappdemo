Important Keyword

The important keyword signals browser that the CSS attribute must override everything else.

The below button has green background originally, but a CSS attribute with the important keyword makes this pink

CSS Code for this Page


        button {
            background: pink !important;
        }

        #button {
            background: green;
        }
