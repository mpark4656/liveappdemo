Flexbox Direction

Flexbox is a one-dimensional layout method for laying out items in rows or columns. It's a recent addition to css to address common layout frustrations. Flexbox allows us to distribute space dynamically across elements of an unknown size.

Flex Direction

Flex is enabled for a container by setting the display attribute to "flex". The default direction of flex is "row", which means the items are placed on a horizontal row, left to right.
