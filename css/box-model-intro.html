CSS Box Model Introduction

Width and Height

Every element is in a box in HTML. Width and Height are the attributes that control the size of the content area.

The below box has a visible border to represent the box that the text contents are in. The colored area indicates the inner content area. The width and height have been set to 200px.

Border

Three important properties of border are width, color, and style.

The below box has a 3px width, black color, and dotted-style border. All three properties of the border can be short-handed with just the "border" attribute. Ex) border: black 1px solid;

You can also style just one side of the box. The box below has only the left side styled and visible.

Border's corners can be rounded with the border-radius attribute.

Box-Sizing

With the default box-sizing (content-box), the inner content's width is maintained regardless of the size of the border. Even with the thickest border, the inner width will always be whatever width value is set. With the border-box property, the width of the border is counted in the width of the entire box.

The two boxes below have the same width of 200px. Due to the thickness of the border, the overall sizes of the boxes are different. The second box has the border-box sizing property, so the width of that box is exactly 200px while the first box's width is a bit larger than 200px because of the additional thickness of its border.
