Specificity of CSS

General Rule of Thumb: The more specific a selector is, the more likely that it will win over less specific selector referencing the same element.

The paragraph below is given two different colors through two separate CSS selectors: One selector is using the paragraph's id while the other selector is referencing the paragraph through the class. The selector using an id is more specific than a class, so the id wins and the background color of green is applied.

This paragraph will probably have a green background.

The general order of specificity is id > class > element (tag)
