CSS Display

Inline elements fit in alongside other elements. Block level elements take up a while "block" of space. Using the display attribute, you can change a block element to inline element and vice versa.

  • INLINE: Width & Height are ignored. Margin & Padding push elements away horizontally but not vertically
  • BLOCK: Block elements break the flow of a document. Width, Height, Margin, & Padding are respected.
  • INLINE-BLOCK: Behaved like an inline element except Width, Height, Margin, & Padding are respected.

Span elements do not respect vertical margin or padding. The two spans below have vertical margin set to 10px and yet, you don't see a margin between them.

If you change the spans to INLINE-BLOCK display, they will respect the vertical margin/padding. Now, you see the vertical margin between the two spans.
