Flexbox - Justify Content

Justify-content behaves differently depending on whether the flex direction is row or column. flex-start is the default value for justify-content.

For the row direction, flex-start puts all elements on the left side of the row. Flex-end puts all elements on the right of the row. Center puts all elements in the center of the row. Essentially, justify content aligns items on the horizontal axis for the row flex direction.

Space-between takes all space and distribute it between the elements but on the outside edges. Space-around gives each element the same amount of space around it. Space-evenly gives each element the same amount of space around it as well as between the elements and the container.
