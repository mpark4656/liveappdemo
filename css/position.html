Position

There are 5 positions. Position attribute can be used with top, right, bottom, left, and z-index except the static position, which is not impacted.

  • static
  • relative
  • absolute
  • fixed
  • sticky

static is the default value for the position. All elements are static unless you specify different position values

Static Position

For static value, top, right, bottom, left, and z-index attributes have no impact.

Relative Position

top, right, bottom, left, and z-index will offset the element relative to itself. The middle box below has been offset with the top and left attributes.

Absolute Position

The element is removed from the normal document flow (No space is created for it). It's position is relative to the closest positioned ancestor (Closest element that is positioned except static).

Fixed Position

The fixed element is removed from the normal document flow and no space is created for the element in the page. It is positioned relative to the initial containing block established by the viewport, except when one of its ancestors has a transform, perspective, or filter property set to something other than none. The final position of a fixed element is determined by the values of top, right, bottom, and left.

Fixed Position

Sticky

Similar to fixed: It's not fixed when it's in a full view, but once you scroll past the element, it will be fixed and "stick" to the containing block.
